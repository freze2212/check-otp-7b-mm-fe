## Check Đại Lý - Landing Page (Base Vite)

Dự án này dùng **Vite + React + TypeScript** để build một trang **landing page** đơn giản cho Check Đại Lý.

Hiện tại mới chỉ dựng **base project** và để **trang chủ là trang trắng**, sẵn sàng để bạn bắt đầu thiết kế UI.

### Cấu trúc cơ bản
- `src/main.tsx`: mount React vào DOM
- `src/App.tsx`: component trang chủ (hiện đang là trang trắng)
- `public/`: chứa static assets

### Cách chạy project
- Cài dependency (nếu chưa cài):
  - `npm install`
- Chạy dev server:
  - `npm run dev`
- Build production:
  - `npm run build`
- Preview build:
  - `npm run preview`

Bạn có thể bắt đầu thiết kế landing page bằng cách sửa file `src/App.tsx` và thêm component/section theo ý tưởng của bạn.
